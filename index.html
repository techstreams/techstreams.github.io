Tech "Streams"

A project stream by @techstreams

  • TSTranslateKhmer

    2014-09-26

    Demonstration Google Apps Script which translates Cambodian Khmer to English through a Google Form using Google Translate. An email containing both the Khmer text and English translation is automatically sent to the form owner each time a user submits to the form.

    Docs Github

  • TSContactForm

    2014-09-08

    Google Apps Script for Google Forms which automatically emails the form owner each time a user submits a response to the form. Emails are nicely formatted and contain the contents of the form submission.

    Docs Github

  • TSMarkMail

    2014-08-09

    Google Drive Add-on for Google Docs which provides an easy way to create email with markdown and send it directly from a Google Document.

    Docs Slides

  • MarkIt Web Publisher

    2012-09-10

    A Google Apps Script web publishing tool for Google Drive. Includes a built-in editor for markdown content upload, creation and edit along with an easy interface for hosting, configuring and deploying that content to the web.


    Semi-Finalist in the 2012 Google Apps Developer Challenge.
